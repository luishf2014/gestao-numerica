# Contexto do Projeto

Esta aplicação é uma plataforma configurável para gestão de concursos numéricos.

O sistema NÃO é um jogo fixo, mas um motor genérico que permite:
- Criar concursos com regras configuráveis
- Definir universo numérico
- Executar múltiplos sorteios
- Calcular ranking e rateios automaticamente

Tecnologias:
- Frontend: React + Vite + Tailwind
- Backend: Supabase (Postgres + Auth + Edge Functions)
- Pagamentos: Pix (Asaas)

O foco do projeto é:
- Clareza de regras
- Rastreabilidade de dados
- Escalabilidade
