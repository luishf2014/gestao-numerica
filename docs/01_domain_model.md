# Modelo de Domínio (Conceitual)

## Usuário
Representa qualquer pessoa no sistema.
Pode ser administrador ou participante.

## Concurso
Objeto central do sistema.
Define regras, datas, universo numérico e status.

## Participação
Vínculo entre usuário e concurso.
Contém números escolhidos, status e pontuação atual.

## Sorteio
Evento que ocorre dentro de um concurso.
Possui data, hora e números sorteados.

## Pagamento
Evento financeiro associado a uma participação.
Define valor, status e método.
